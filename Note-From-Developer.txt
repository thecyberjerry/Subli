# Subli
This is a Basic Python Script to scan for Subdomains of provided target
Give it a target and it will scan for the subdomains.
This Script uses subdomain list to scan available active subdomains so if you want to add custom subdomain to your enumeration list add the custom word to list.txt provided with this python script or simply replace your custom wordlist with default wordlist came with this python script. 
If using list other than default provided than paste the list file in this python file directory renaming file to list.txt

This python file is for educational purpose and for penetration testers who want to learn the enumeration in pentesting.
